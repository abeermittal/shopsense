# ShopSense — SDC × GDG Recruitment Submission

ShopSense is a modern, dark-themed e-commerce web app built using HTML, CSS, and JavaScript. 
It demonstrates product browsing, search, add-to-cart functionality, cart persistence using localStorage, and a checkout simulation.

## How to run locally
1. Clone the repo:
   git clone [https://github.com/<your-github-username>/shopsense.git](https://github.com/abeermittal/shopsense.git)
2. Open terminal in the project folder and run:
   python -m http.server 5500
3. Open http://localhost:5500 in your browser.

## Live demo
shopsense-git-main-abeermittals-projects.vercel.app

## Tech stack
- HTML, CSS, JavaScript
- Local JSON (`products.json`) used as product data source
- localStorage for cart persistence
